# QueRule

Juego de papelitos en un solo dispositivo.

## Que es

QueRule es una app movil para jugar por equipos pasando el mismo telefono entre jugadores.

## Flujo de juego

1. Crear partida.
2. Configurar tiempo, rondas y palabras por jugador.
3. Cargar nombres de jugadores y equipo.
4. Cada jugador agrega sus palabras cuando recibe el telefono.
5. Comienza la partida por turnos en la misma pantalla.

## Requisitos

- Node.js 18 o superior
- npm
- Expo Go (opcional para probar en telefono)

## Ejecutar

```bash
cd mobile
npm install
npx expo start
```
