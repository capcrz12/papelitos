# Mobile Source

Codigo fuente de la app movil QueRule.

## Estructura

```text
src/
  components/
  constants/
  hooks/
```

## Notas

- `hooks/useGestureDetector.ts`: deteccion de gesto para interaccion durante el turno.
- `components/`: componentes reutilizables de interfaz.
- `constants/theme.ts`: colores y tokens visuales.
