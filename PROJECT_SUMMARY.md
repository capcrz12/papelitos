# Project Summary

QueRule es una aplicacion movil para jugar papelitos por equipos en un solo dispositivo.

## Estado actual

- Flujo completo implementado en frontend.
- Configuracion de partida y equipos.
- Carga de palabras por jugador.
- Juego por turnos con temporizador y puntaje.

## Archivos clave

- `mobile/app/index.tsx`
- `mobile/app/config.tsx`
- `mobile/app/word-submission.tsx`
- `mobile/app/game.tsx`

## Stack

- React Native
- Expo
- TypeScript

## Siguiente enfoque

- Mejoras de UX en transiciones.
